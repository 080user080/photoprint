# PhotoPrint

Програма для автоматичної обробки та друку фотографій та документів з пакетною обробкою.

## Функціональність

- **Автоматична класифікація документів:** чорно-білі, кольорові документи, фото
- **Інтелектуальна авто-обробка:**
  - Авто-видалення градієнтних тіней (background estimation: blur/divide)
  - Авто-яскравість на основі гістограми (5-95% перцентилів)
  - Авто-контраст з адаптивними порогами
  - Авто-різкість на основі аналізу blur (Laplacian variance)
  - Авто-детекція перспективи з fallback
- **Ручне коригування:**
  - Слайдери: Тіні, Яскравість, Контраст, Різкість, HDR
  - Кнопки: Ч/Б, Авто-перспектива, Ручна перспектива
  - Скидання: Скинути перспективу, Скинути слайдери
- **Пакетна обробка:** Авто та ручний режими для черги файлів
- **Drag & Drop:** Підтримка перетягування файлів та папок (Windows)
- **Запам'ятовування налаштувань:** Ручні корекції зберігаються для кожного файлу
- **Друк:** Відправка на принтер (priPrinter або інший)
- **Збереження:** Опціональне збереження оброблених файлів з суфіксом `_edited`

## Використання

### Базовий робочий процес

1. **Додайте файли:** Перетягніть файли у чергу або використайте кнопки "Додати файли" / "Додати папку"
2. **Оберіть режим:** **Авто** або **Ручний**
3. **Авто режим:**
   - Натисніть "Друкувати все"
   - Програма автоматично класифікує, обробить і надрукує всі файли
4. **Ручний режим:**
   - Переглядайте кожне фото по черзі
   - Використовуйте "Auto Fix" для автоматичної корекції
   - Або налаштуйте вручну через слайдери
   - Натисніть "Друк" або "Пропустити"

### Слайдери

- **Тіні (0.0-2.0):** Висвітлення темних областей
- **Яскравість (-1.0-1.0):** Зміна загальної яскравості
- **Контраст (-1.0-1.0):** Зміна контрасту
- **Різкість (0.0-1.0):** Збільшення різкості
- **HDR (0.0-1.0):** Витягування деталей з тіней та світл

### Кнопки авто-корекції

- **Авто яскравість:** Автоматичне вирівнювання яскравості на основі гістограми
- **Авто контраст:** Автоматичне вирівнювання контрасту
- **Авто різкість:** Автоматична різкість на основі аналізу blur

### Перспектива

- **Авто-перспектива:** Автоматичне знаходження та виправлення перспективи документа
- **Ручна перспектива:** Ручне встановлення 4 кутів документа
- **Скинути перспективу:** Повернення до оригінальної перспективи

### Налаштування

Файл `config/settings.ini` — текстовий, редагується блокнотом.
Або через кнопку **⚙ Налаштування** у програмі.

**Доступні налаштування:**
- Пороги класифікації документів
- Сила авто-різкості та поріг blur
- Параметри авто-яскравості/контрасту (перцентилі)
- Налаштування бінаризації для ч-б документів
- Якість JPG при збереженні
- Папка для збереження оброблених файлів
- Назва принтера за замовчуванням
- Вікно програми (розмір, позиція)
